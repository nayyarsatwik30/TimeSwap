# ⏳ TimeSwap — Skill Exchange Platform

> *Turning time into currency. Learn. Teach. Grow.*

---

## 🚀 Overview

**TimeSwap** is a modern cross-platform application designed to enable **peer-to-peer skill exchange** using a **time-credit system**. Instead of paying money, users trade time — teach a skill, earn credits, and use those credits to learn something new.

This project is built with a **Flutter frontend** and a **scalable backend architecture**, aiming to simulate a real-world product with clean UI flows and modular system design.

---

## 🎯 Core Idea

* 💡 Everyone has something valuable to teach
* ⏱️ Time becomes the unit of exchange
* 🌐 Learning becomes decentralized and community-driven
